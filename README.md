# Library Management System

A Java console application for managing a library system using Object-Oriented Programming principles.

## Features

- Register books
- Register members
- Manage loans
- Book categories using enums
- Input validation
- Menu-based console interface

## Technologies Used

- Java
- IntelliJ IDEA
- Git
- GitHub

## Project Structure

src
├── model
├── service
├── enums
├── ui
├── util
└── main

## Concepts Applied

- Object-Oriented Programming (OOP)
- Encapsulation
- Separation of concerns
- Enums
- Basic architecture organization
